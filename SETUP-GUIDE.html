ทำเสร็จกี่ขั้น? 0/5

1
หมุน Supabase Key
⏱ 3 นาที

ขั้นตอนนี้ทำให้ password เก่าใช้ไม่ได้ — กันคนอื่นเข้าระบบคุณ

ขั้นที่ 1: คลิกลิงก์นี้เปิด Supabase Dashboard
→ เปิด Supabase Dashboard
ขั้นที่ 2: เลือกโปรเจคของคุณ → คลิก Settings → API
ขั้นที่ 3: หาหัวข้อ "service_role key" → คลิก Regenerate → ยืนยัน
ขั้นที่ 4: กดปุ่ม Copy เพื่อคัดลอก key ใหม่
⚠️ เก็บ key นี้ไว้! ต้องเอาไปใส่ใน Vercel (ขั้นตอนถัดไป)
  • เปิด Supabase Dashboard แล้ว
  • หา service_role key เจอแล้ว
  • กด Regenerate แล้ว
  • คัดลอก key ใหม่แล้ว
2
แก้ Key ใน Vercel
⏱ 2 นาที

เอา key ที่คัดลอกมาใส่ใน Vercel เพื่อให้เว็บใช้ key ใหม่

ขั้นที่ 1: คลิกลิงก์นี้เปิด Vercel Dashboard
→ เปิด Vercel Dashboard
ขั้นที่ 2: คลิกโปรเจค music-marketplace → Settings → Environment Variables
ขั้นที่ 3: หา SUPABASE_SERVICE_ROLE_KEY → คลิก Edit → วาง key ใหม่ → กด Save
ขั้นที่ 4: ทำเหมือนกันกับ JWT_SECRET (ถ้ามี)
  • เปิด Vercel Dashboard แล้ว
  • เจอ Environment Variables แล้ว
  • แก้ SUPABASE_SERVICE_ROLE_KEY แล้ว
  • กด Save แล้ว
3
ทดสอบจ่ายเงินจริง
⏱ 3 นาที

ทดสอบว่า PromptPay QR ทำงานได้จริง

ขั้นที่ 1: คลิกลิงก์นี้เปิดเว็บ BEAT VAULT
→ เปิด BEAT VAULT
ขั้นที่ 2: Login ด้วย email/password ของคุณ
ขั้นที่ 3: ไปที่ Browse → เลือก track ที่มีราคา (เช่น $0.50-$0.90) → กด Add to Cart
ขั้นที่ 4: กด Checkout → เลือก PromptPay QR
ขั้นที่ 5: เปิด app ธนาคาร → สแกน QR → โอนเงินตามจำนวนที่แสดง
ขั้นที่ 6: กลับมากดปุ่ม "ฉันได้ชำระเงินแล้ว"
✅ สำเร็จ! ถ้า track ปลดล็อคให้ download = ระบบทำงานถูกต้อง
  • เปิดเว็บแล้ว
  • Login ได้แล้ว
  • เลือก track + Add to Cart แล้ว
  • QR แสดงแล้ว
  • โอนเงินแล้ว
  • Download ได้แล้ว
4
ลบ .env จาก Git History
⏱ 1 นาที

ลบ password ที่เคย commit ไว้ใน git

เปิด Terminal แล้วรันคำสั่งนี้:
brew install bfg
bfg --delete-files .env
git push origin main --force
⚠️ คำสั่งนี้จะ force push — ถ้ามีคนร่วมโปรเจคอยู่ ต้องระวัง
  • เปิด Terminal แล้ว
  • รัน brew install bfg แล้ว
  • รัน bfg --delete-files .env แล้ว
  • รัน git push --force แล้ว
